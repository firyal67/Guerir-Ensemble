# 🎀 Guérir Ensemble

**🌐 Site en ligne : [https://guerir-ensemble.feryelguehis86.workers.dev](https://guerir-ensemble.feryelguehis86.workers.dev)**

Un site web de sensibilisation au cancer du sein, visant à informer, soutenir et encourager le dépistage précoce.

## Pages
- **Accueil** — Statistiques, missions, médecins partenaires, événements
- **Ressources** — Quiz éducatif, calculateur de dépistage, infographies, vidéos
- **Inscription** — Création de compte avec validation en temps réel
- **Connexion** — Authentification utilisateur

## Stack technique
- HTML5 / CSS3 / JavaScript vanilla
- Bootstrap 5.3
- Déployé sur Cloudflare Workers
